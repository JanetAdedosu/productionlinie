# Production Line Simulation

This project simulates a simple 3-machine production system using Python.

## Description
The system consists of:
Machine A → Machine B → Machine C

The simulation analyzes:
- Waiting times
- Bottlenecks
- Production efficiency

## Key Result
The analysis showed that Machine C is the bottleneck due to higher waiting times.

- Python
